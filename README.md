# E-commerce-Sales-Dashboard

Interactive Power BI dashboard for e-commerce sales analysis.

## Overview 
- This project focuses on transforming raw sales data into meaningful insights using data visualization. It helps businesses track perfomance and make informed decisions.

## Objectives 
- Monitor total sales performance.
- Identify top selling products and categories.
- Analyze customer demographics.
- Understand payment method preferences.

## Key Performance Indicators (KPIs) 
- Total Sales Revenue.
- Total Quantity Sold.
- Top Perfoming Products.
- Sales Distribution by Country.
- Payment Method Analysis.

## Tools Used 
- Microsoft Excel
- Power BI.

## Insights 
- Identified best-selling  products.
- Revealed Customer purchasing patterns.
- Highlighted sales trends across different countries.
- Showed the preferred payment methods.
